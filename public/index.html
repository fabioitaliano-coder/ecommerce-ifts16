Clase 8, 9, 10 y 11

Catálogo con carrito, seguridad y panel administrador real

Esta pantalla consume la API real, aplica filtros por categoría con query parameters, usa relaciones de Sequelize para mostrar la categoría de cada producto, persiste el carrito en localStorage, completa el checkout, adapta la interfaz según la sesión y ahora permite mantenimiento administrativo de entidades.

Recursos conectados

  • GET /api/productos
  • GET /api/categorias
  • GET /api/clientes
  • GET /api/descuentos
  • POST /api/checkout
  • POST /api/auth/login
